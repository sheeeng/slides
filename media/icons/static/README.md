# Static Icons

[Noto Emoji](https://github.com/googlefonts/noto-emoji) repository.

## Source

PNG base URL: `https://raw.githubusercontent.com/googlefonts/noto-emoji/main/png/512/emoji_u{codepoint}.png`

SVG base URL: `https://raw.githubusercontent.com/googlefonts/noto-emoji/main/svg/emoji_u{codepoint}.svg`
